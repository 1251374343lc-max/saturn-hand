🪐 Saturn Echo: 基于手势识别的实时交互 3D 粒子系统
1. 项目简介 (Project Overview)
Saturn Echo 是一款融合了计算机视觉（Computer Vision）与 WebGL 3D 渲染技术的沉浸式网页应用。该项目利用摄像头实时捕捉用户手部动作，驱动数万个 3D 粒子在“土星形态”与“星系爆发”之间流畅切换。

不仅限于视觉特效，项目还通过随机生成的温馨短语与空灵的背景音乐，构建了一个兼具科技感与人文浪漫的数字化交互空间。

2. 核心功能 (Core Features)
👋 实时手势交互 (Real-time Gesture Control)
利用 MediaPipe 先进的手部追踪技术，系统能毫秒级精准识别用户手势：

握拳 (Fist)：引力凝聚。数万个粒子受到引力牵引，平滑汇聚成一颗旋转的金色土星及其蓝色星环。

张手 (Open Palm)：大爆炸 (Big Bang)。粒子瞬间失去引力束缚，向宇宙深处无序扩散，模拟星系爆发的壮丽景象。

🌌 影院级视觉特效 (Cinematic Visuals)
高精度粒子系统：基于 Three.js 开发，渲染超过 15,000 个动态粒子。

数学几何建模：通过球坐标与圆环算法精确还原土星结构，叠加多层噪点与色彩渐变。

沉浸式环境：包含深邃的宇宙雾效、动态光照以及粒子呼吸动画。

💌 情感化交互体验 (Emotional Engagement)
浪漫短语生成：当粒子散开时，屏幕中央会随机浮现一句治愈系短语（内置 100 句精选文案，如“星河滚烫，你是人间理想”），为交互增添情感温度。

视听同步：集成了空灵的宇宙氛围音乐（Ambient Space Music），配备可视化播放控制与毛玻璃风格 UI。

3. 技术栈 (Tech Stack)
核心引擎：Three.js (WebGL 3D 渲染)

计算机视觉：Google MediaPipe Hands (手势识别)

前端基础：HTML5 / CSS3 (响应式布局 & 毛玻璃 UI)

语言：JavaScript (ES6+)

4. 操作说明 (User Guide)
第一步：启动与授权
打开项目网页链接。

浏览器会弹出提示，请点击**“允许” (Allow)** 使用摄像头权限（用于手势捕捉，画面仅在本地处理，不会上传）。

点击页面任意位置或右下角的**“播放音乐”**按钮，开启沉浸式背景音。

第二步：手势控制
请确保手掌处于摄像头画面中央，距离屏幕约 30-50 厘米：

✊ 凝聚土星：

动作：握紧拳头。

效果：粒子将迅速从四面八方汇聚，重组为土星形态，文字消失。

🖐️ 释放星系：

动作：五指张开。

效果：土星“爆炸”，粒子向四周飘散，屏幕中央浮现一句随机的浪漫短语。

第三步：辅助功能
全屏模式：点击右下角的“全屏沉浸”按钮，隐藏浏览器边框，获得最佳体验。

音乐控制：点击音符按钮可暂停/播放背景音乐。

5. 兼容性说明 (Compatibility)
桌面端 (PC/Mac)：推荐使用 Chrome、Edge 或 Safari 浏览器，体验最佳。

移动端 (Mobile/Tablet)：支持大部分现代手机浏览器。

注意：在移动设备上，系统会自动调用前置摄像头。请保持手机竖屏或横屏锁定，并将手掌置于前置镜头可视范围内。

6. 开发与部署 (Development & Deployment)
本项目纯前端实现，无需后端服务器。

本地运行：需使用 VS Code 的 Live Server 插件启动（因浏览器安全策略，直接打开 HTML 文件无法加载摄像头）。

在线部署：支持直接部署至 GitHub Pages、Vercel 或 Netlify，需确保环境支持 HTTPS 协议以获取摄像头权限。
